## 🔄 LANGKAH DEPLOY:

```bash
# Add semua file
git add .

# Push ke GitHub
git push origin main
```

### STEP 2: Deploy ke Vercel

**Opsi A: Via Website (Recommended)**
1. Buka https://vercel.com
2. Login dengan akun GitHub
3. Klik "Add New..." → "Project"
4. Pilih repository `akbary-afk/web-tralis-` atau `akbary-afk/integrasi-web`
5. Vercel otomatis detect `vercel.json` ✓
6. Build Settings:
   - Framework: None (custom)
   - Root Directory: ./
   - Build Command: (kosongkan)
   - Install Command: pip install -r requirements.txt
7. Klik "Deploy" ✓

**Opsi B: Via CLI (jika sudah install Vercel)**
```bash
npm install -g vercel
cd web-integrasi-pro
vercel --prod
```
